# V0.8-fix ブランチ 段階マージ・検証計画

## 背景

`v0.7.0` → `feature/v0.8-features` の間に 8 コミットが積まれているが、
何かが壊れている可能性がある。  
`v0.8-fix` ブランチを `v0.7.0` から切り、1 コミットずつ cherry-pick → ビルド・テスト検証を行い、
問題のあるコミットを特定する。

## リスク分析

- **#1〜#5** は docs / spec のみ → ビルド・動作に影響なし。まとめて取り込んでも安全。
- **#6 (`a038e88`)** — `pdfBookmarks.ts` の PDFString → PDFHexString 変更。小さいが PDF 出力に直接影響。
- **#7 (`db8958a`)** — preview.js / preview.css / webviewPanel.ts を大きく変更。プレビュー表示に影響。
- **#8 (`3acfc42`)** — 44 ファイル変更の大型コミット。`extension.ts`, `dependencyManager.ts`, `presets.ts`, `renderPlantUml.ts`, `exportPdf.ts`, `package.json`, `tsconfig.json` など広範囲。**最も壊れやすい。**
