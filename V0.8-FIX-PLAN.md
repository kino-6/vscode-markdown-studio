## 追加修正 (E2E で発見)

### ズーム状態保持バグ
- **症状**: ダイアグラムをズームした後、操作を止めると約1秒後に100%に戻る
- **原因**: `rerenderMermaid` が高解像度 SVG 再生成後に `mermaidHost.style.transform = 'none'` でリセットしていた。SVG の `max-width: 100%` 制約で拡大サイズがコンテナ幅に制限され、元サイズに戻っていた
- **修正**: re-render 後に `applyTransform(container, state)` を呼んで CSS transform を再適用。`svg.style.maxWidth = 'none'` で制約を解除。`update-body` 時の zoom state 保存・復元も追加

### markdown-pdf テーマ blockquote
- **変更**: グレーボーダー → 緑ボーダー (#4caf50) + 薄い背景。markdown-pdf 拡張の見た目に合わせた

### デフォルトテーマ変更
- **変更**: `markdownStudio.style.theme` のデフォルトを `"default"` → `"markdown-pdf"` に変更。markdown-pdf ユーザーが移行しやすいように
