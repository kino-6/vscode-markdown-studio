# バグ修正要件ドキュメント

## はじめに

`addBookmarks()` 関数で生成されるPDFブックマーク（アウトライン）において、日本語の見出しタイトルが文字化け（mojibake）して表示されるバグの修正。根本原因は、`pdf-lib` の `PDFString.of()` がLatin-1文字のみをサポートするPDFDocEncodingを使用しているため、日本語などの非ASCII文字が正しくエンコードされないことにある。UTF-16BEエンコーディングを使用する `PDFHexString` に切り替える必要がある。

## バグ分析

### 現在の動作（不具合）

1.1 WHEN ブックマークタイトルに日本語文字（例：「1. マークダウンレンダリング」）が含まれる場合 THEN PDFビューアのブックマークサイドバーに文字化けしたテキストが表示される

1.2 WHEN ブックマークタイトルに中国語・韓国語・アラビア語などの非ASCII文字が含まれる場合 THEN PDFビューアのブックマークサイドバーに文字化けしたテキストが表示される

1.3 WHEN ブックマークタイトルに絵文字（例：「📝 メモ」）が含まれる場合 THEN PDFビューアのブックマークサイドバーに文字化けしたテキストが表示される

### 期待される動作（正しい動作）

2.1 WHEN ブックマークタイトルに日本語文字（例：「1. マークダウンレンダリング」）が含まれる場合 THEN PDFビューアのブックマークサイドバーに日本語テキストが正しく表示されるものとする

2.2 WHEN ブックマークタイトルに中国語・韓国語・アラビア語などの非ASCII文字が含まれる場合 THEN PDFビューアのブックマークサイドバーに該当テキストが正しく表示されるものとする

2.3 WHEN ブックマークタイトルに絵文字（例：「📝 メモ」）が含まれる場合 THEN PDFビューアのブックマークサイドバーに絵文字を含むテキストが正しく表示されるものとする

### 変更なしの動作（リグレッション防止）

3.1 WHEN ブックマークタイトルがASCII文字のみ（例：「1. Introduction」）で構成される場合 THEN ブックマークサイドバーにテキストが引き続き正しく表示されるものとする

3.2 WHEN ブックマークタイトルにASCIIと非ASCII文字が混在する場合（例：「1. はじめに - Introduction」） THEN ブックマークサイドバーにテキスト全体が引き続き正しく表示されるものとする

3.3 WHEN ブックマークツリーの階層構造（親子関係、兄弟リンク）が構築される場合 THEN ブックマークの階層構造が引き続き正しく維持されるものとする

3.4 WHEN ブックマークのページ遷移先（/Dest）が設定される場合 THEN 各ブックマークが引き続き正しいページにリンクされるものとする
